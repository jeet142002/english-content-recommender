# Product Flow

## Screens

### Landing

- one dominant CTA: `Recommend me something great`
- premium minimal dark presentation
- clear promise: English-only movies and series
- short copy about rating a few titles

### Priming

Two lightweight choices:

- `Movie / Series / Either`
- `Safe picks / Mix it up / Surprise me`

This sets a prior without turning the product into a survey.

### Rating loop

Each title card exposes:

- poster
- title and metadata
- short synopsis
- tone and style tags
- actions: `Like`, `Dislike`, `Not seen`, `Stop and recommend`

### Recommendation reveal

- one hero recommendation
- confidence signal
- explanation reasons
- backup options
- restart action

## Session events

Each response should be logged with:

- `sessionId`
- `titleId`
- `value`
- `step`
- timestamp

## Quality goals

- early perceived value
- low fatigue
- high recommendation trust
- visible difference between `not seen` and `dislike`
